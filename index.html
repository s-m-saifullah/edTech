CSS Grid and Flexbox are layout models that have similarities and can be used together. The main dissimilarity is that CSS Grid can be used to create two-dimensional layouts, while Flexbox can only be used to create one-dimensional layouts. That means one can place components along the X- and Y-axis in CSS Grid and only one axis in Flexbox.

In CSS, the term Box Model is used when we are talking about the design and layout of a page. The CSS Box Model is essentially a box that wraps around every HTML element. It consists of Margins, Borders, Padding, and the Actual content of the web page.

The HTML semantics refers to the tags that provide meaning to an HTML page rather than just presentation. It makes HTML more comprehensible by better defining the different sections and layout of web pages. Some semantic tags are as follows:
  • <header>: It defines a header for a web page.
  • <nav>: It defines a container for navigation links.
  • <section>: This defines a section in a web page.
  • <article>: This element contains the main part, containing information about the web page.
  • <aside>: The <aside> content is often placed as a sidebar in a document.
  • <footer>: It defines a footer for a document or a section.
